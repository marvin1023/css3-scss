# css3 scss

在这推荐使用[autoprefixer](https://github.com/postcss/autoprefixer)来智能生成前缀（有grunt和gulp版本，有些客户端编译软件也集成了这个功能）。

该文件之所以独立出来，是为了那些不想使用上面方法的人而准备的方案。

使用说明，请参考[css3 scss使用说明](http://www.w3cplus.com/sasscore/css3.html)
